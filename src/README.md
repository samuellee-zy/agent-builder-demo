# ⚛️ Frontend Application

This directory contains the React source code for the Agent Builder.

## Structure
- **`components/`**: Reusable UI components (AgentBuilder, ToolsLibrary, etc.).
- **`services/`**: Frontend business logic (Orchestrator, Tool execution, Storage).
- **`App.tsx`**: Main application component.
- **`types.ts`**: Shared TypeScript definitions.
